# ⏱️ Timer & Stopwatch Web App

A simple, responsive, and modern **Timer & Stopwatch Web Application** built using **HTML, CSS, and JavaScript**. It features a clean UI, smooth animations, and accurate time tracking.
## 🚀 Features
* Start, Pause, and Reset Timer
* Stopwatch with Lap Functionality
* Modern and Clean UI Design
* Fully Responsive (Mobile + Desktop)
* Smooth Animations and Transitions
* Accurate Time Tracking

## 🛠️ Technologies Used

* HTML5 – Structure
* CSS3 – Styling & Responsive Design
* JavaScript (Vanilla JS) – Functionality
